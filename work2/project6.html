Johns Hopkins University | EN.530.666 Magnetically Actuated Robots

Haptic-Guided Magnetic Microbot Navigation

Spring 2025

Developed a real-time wall collision detection system with gradient haptic feedback for magnetically actuated surgical microrobots. The system helps operators “feel” walls that are otherwise hard to perceive visually during teleoperation.

My contribution: collision detection algorithm using joystick-robot direction dot product, custom maze design and fabrication, and a 4-user haptic feedback study.

Problem Statement

Magnetic microbots can navigate in confined environments, but operators often struggle to detect wall contact early enough to avoid collisions. Because the microrobot is small and the environment is visually dense, a teleoperator needs additional cues beyond the camera feed. This project explores haptic feedback as an intuitive way to indicate when the robot is approaching or touching a wall.

Methodology

The system combines electromagnetic actuation, visual localization, and joystick-based haptic feedback. Robot position is tracked from an overhead camera feed, then compared against the maze geometry to estimate collision risk. When the direction of intended motion conflicts with the detected wall proximity, the joystick vibrates with intensity that increases as the robot gets closer to the wall.

The main control architecture was implemented in MATLAB, with helper scripts handling localization, wall detection, actuation, and vibration control.

Design & Simulation

The core collision logic uses the dot product between the joystick command direction and the robot-to-wall direction. If the robot is moving toward a wall and the geometric conditions indicate imminent contact, the system triggers haptic vibration. This creates a “virtual tactile boundary” for the operator by utilizing the haptic motor present in modern joysticks. The vibration is directly proportional to the force dot product.

Fabrication

A custom test maze was fabricated for controlled navigation experiments. The maze was designed to create narrow passages and turning regions where wall detection and haptic feedback would be useful. The setup also included the actuation hardware, joystick interface, and camera-based localization pipeline.

Test Setup

The setup includes a 4-coil electromagnetic array, Arduino microcontroller, dual motor controllers, joystick interface, overhead camera, and a laptop running MATLAB for actuation, localization, and feedback control.

Results

The navigation interface successfully displayed wall proximity in real time and provided haptic cues through the joystick. During demonstration, the robot was able to traverse the maze while the operator received feedback whenever the robot approached a wall.

The live overlay showed wall detection status, frame rate, and robot position coordinates, allowing the operator to maintain situational awareness while relying less on direct visual inspection of the maze boundaries.
