  "project_summary": "This project developed a haptic feedback system for magnetically actuated surgical microrobots, enabling operators to 'feel' when the robot collides with obstacles without relying on complex image recognition. The system uses a 4-coil electromagnetic array to control a capsule-shaped robot in a water-filled petri dish, with overhead camera localization tracking red/blue color markers at ~5 Hz frame rate.\n\nMy primary contribution was implementing the collision detection algorithm, which compares the joystick input direction against the robot's actual movement using vector dot products. When the robot attempts to move but displacement remains below threshold (indicating a wall), the system triggers gradient haptic feedback that intensifies over time – allowing operators to distinguish between grazing contact and sustained collision.\n\nI also designed and fabricated the physical maze used for testing, and facilitated user studies with 4 participants. Results showed users could successfully navigate the maze using haptic cues alone, with potential clinical applications for navigating surgical robots through intestines or blood vessels where camera feedback may be delayed or unavailable.",
