  "project_summary": "Developed an MRI-compatible robotic system for image-guided needle insertion into lumbar spine intervertebral spaces (L1-L5) as part of a graduate robotics course at Johns Hopkins University. The robot features a 3-stage pneumatic actuation system: Stage 1 for Y-axis translation, Stage 2 for angular rotation (theta), and Stage 3 for needle insertion depth. The control pipeline processes MRI images through 3D Slicer for slice selection and window-leveling, then uses MATLAB for vertebrae boundary detection via morphological operations and centroid computation. An optimal needle path is calculated by maximizing distance from vertebrae boundaries. The system converts image coordinates to robot commands through inverse kinematics with step conversion. After identifying and fixing critical bugs including flipped X/Y coordinates, an incorrect conversion factor (1.367 vs. actual 1.20225), and theta reference line errors, the system achieved needle depth accuracy of less than 2mm. This project provided hands-on experience with medical robotics, image-guided intervention, and the unique challenges of MRI-compatible system design.",
